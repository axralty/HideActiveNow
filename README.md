HideActiveNow — BetterDiscord Plugin
A simple BetterDiscord plugin that removes the Active Now panel on the right side of Discord — no more seeing who's in which server, playing what game, or sharing their screen.

Features

Hides the entire Active Now sidebar panel
Lightweight — pure CSS injection, no performance impact
Toggle on/off anytime from the BetterDiscord plugins menu


Requirements

BetterDiscord installed


Installation

Download HideActiveNow.plugin.js
Open Discord
Go to User Settings → BetterDiscord → Plugins
Click Open Plugins Folder
Move HideActiveNow.plugin.js into that folder
Back in the Plugins list, toggle HideActiveNow on
